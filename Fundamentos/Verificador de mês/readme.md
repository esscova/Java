## Verificador de Mês

Criar um programa em Java que verifica o valor de uma variável representando um mês do ano e exibe o nome desse mês.

### Instruções

1. Crie uma variável chamada `mes` para representar um mês do ano (1 para janeiro, 2 para fevereiro, etc.).

2. Use a estrutura `switch` para verificar o valor de `mes`.

3. Com base no valor de `mes`, exiba uma mensagem correspondente ao mês.
