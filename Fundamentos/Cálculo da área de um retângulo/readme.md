## Cálculo da Área de um Retângulo

Criar um programa Java que calcule a área de um retângulo com base em sua largura e altura.

### Instruções

- Crie uma classe chamada `CalculadoraAreaRetangulo`.

- Dentro da classe, declare duas variáveis do tipo `double`: `largura` e `altura`.

- Atribua valores às variáveis `largura` e `altura` (por exemplo, 5.0 e 3.0, respectivamente).

- Calcule a área do retângulo usando a fórmula:

- Área = largura × altura

- Exiba o resultado na tela com uma mensagem informativa.

### Não esqueça:

- Salve o arquivo como `CalculadoraAreaRetangulo.java`

- Compile o arquivo com o comando `javac CalculadoraAreaRetangulo.java`

- Execute seu programa usando o comando `java CalculadoraAreaRetangulo`
